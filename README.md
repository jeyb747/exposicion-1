 Usabilidad Web

Proyecto web educativo que explica el concepto de usabilidad web, su importancia, características y ejemplos visuales. Está desarrollado con HTML5, CSS3 y Bootstrap 5.

 Vista general

Este sitio tiene como objetivo enseñar de forma clara qué es la usabilidad web y por qué es fundamental en el desarrollo de páginas modernas. Incluye secciones teóricas y un diseño sencillo y responsivo.
https://jeyb747.github.io/exposicion-1/

 Tecnologías utilizadas
HTML5
CSS3
Bootstrap 5.3.3
Diseño responsive (adaptable a dispositivos móviles)

 Objetivo del proyecto
Explicar qué es la usabilidad web
Mostrar su importancia en el diseño de sitios
Presentar características principales de un sitio usable
Aplicar conocimientos básicos de diseño web
 Secciones del sitio
1. Definición

Explica qué es la usabilidad web y su propósito principal.

2. Importancia

Describe por qué es fundamental para la experiencia del usuario.

3. Características

Enumera los principios de un sitio web bien diseñado.

4. Ejemplo visual

Incluye una imagen representativa del diseño web.

5. Conclusión

Resumen sobre la importancia de la usabilidad.

 Cómo ejecutar el proyecto
Descarga o clona el repositorio
Abre el archivo index.html en tu navegador
Asegúrate de que la carpeta /css e /img estén en la misma ruta
